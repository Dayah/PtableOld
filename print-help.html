How to print this web page

This web page will print much more sharply than the PNG. (Only the actual table will print, not the text, navigation, or ads.)

Internet Explorer

  1. Select Tools, Internet Options. In the Advanced tab under the Print category, check Print background colors and images.
  2. Select File, Page Setup.
    1. For Orientation select Landscape.
    2. Set the Margins to 0. It's fine if they replace themselves with small numbers.
    3. Optional: Remove the text under Header and Footer.
  3. Select File, Print Preview and inspect the layout.
  4. Print

Firefox

  1. Select File, Page Setup.
    1. Select Landscape.
    2. Check Print Background (colors & images).
    3. Under the Margins & Header/Footer tab set all margins to 0.
    4. Optional: Blank out all the Header and Footer fields.
  2. Select File, Print Preview and inspect the layout.
  3. Print
