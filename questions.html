Periodic Table of Elements - Questions

Below you will find the answers to some commonly asked questions about the Periodic Table page. If you have a question that's not on this page or you would like to follow up on one of the below questions, contact me.

General Questions

Who made this periodic table?
Michael Dayah.
Why?
He thought it would look nice converted to HTML and wanted to see if he could do it. He was also bored while staying at home recovering from toe surgery.
When?
September 1997. It was posted on the web October 1, 1997. Element descriptions were added later in December 1997. Descriptions were significantly upgraded in summer 2005 and now use data from Wikipedia in compliance with its license.
What's the big deal? There are other periodic tables on the web.
Not only is this one of the very few tables that has all the information on one page, it's quite possibly the most complex use of HTML tables anywhere on the web. This periodic table was made with approximately seven hundred lines of HTML, no images.
How long did it take to make?
A total of about five hours plus one half hour per month maintenance.
From where was the look of the table taken?
The color scheme is loosely based on a Frank Schaffer Publications poster. The categories were also used from this poster. The idea of using white space between elements was taken from the periodic table in the designer's biology book.
I like your layout. Can I buy a poster?
We're currently looking into producing a periodic table poster. First, we'll need some feedback from you on what size, paper, and finish you'd like.
Can I print it?
Yes, but only for personal or classroom use. Contact me to discuss larger scale distribution.
Will you do my homework?
Only if you're a cute girl. Make sure to attach a picture when you e-mail me.
What accessibility features does this table feature?
Visitors with low vision who use large fonts will find that the table scales as large as necessary while maintaining all contextual information and relative sizing. Despite its picture-like appearance, this table is all text, so visitors with no vision can use software to read or search the page like they would any text document.
Is there a license agreement I must follow?
Because of the frequent piracy of the periodic table, you are not allowed to redistribute the table electronically in any fashion. That means don't save it and upload it to your own web site. Don't worry about my bandwidth; link directly to my images and PDFs if you desire. Of course, you are allowed to print the table out for your own, personal use. If you would like to print out many copies for a classroom, please tell me.
Can I link to your periodic table?

HTML-Related Questions

The table looks distorted or is indecipherable.
The table was made when CSS was barely implemented in browsers. It wasn't until version 5 that it was fully viewable. There is an old browser compatible version, but it is not kept up to date. If you have a modern browser, perhaps style sheets are disabled.
Which browsers can display the table?
Though years ahead of its time in 1997, all modern browsers now display it properly. To an extent, the information can even be extracted with Lynx.
What editor was used to make the periodic table?
Only notepad. No WYSIWYG or enhanced text editors were used. Even four years after its design, no WYSIWYG editors were capable of opening it without clobbering the layout completely.
It could have been done in FrontPage, HotDog, or [insert WYSIWYG HTML editor here].
Probably not. But if it could, it would not be as cross-browser compatible as it is now and would take up at least twice the markup.
What do you mean by "complex tables"?
There are four qualities which make a table complex:
  • The table uses many dependent columns and rows.
  • The structure and markup cannot be optimized any further.
  • COLSPAN and ROWSPAN are used extensively.
  • Removing one cell will likely cause the table to collapse.
But table XYZ is more complex.
If you find a table more complex, please tell me about it.

Visit the "Show Borders" link from the main page.
How deep are the tables nested?

No.
Of which web technologies does the table take advantage?
HTML 4.0, Cascading Style Sheets (level one and two), and JavaScript.
