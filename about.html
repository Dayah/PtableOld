Author:  
       I, Michael Dayah, designed the Periodic Table Page in September 1997 when I was fourteen. I am now sixteen and a senior at McMinn County High School in Athens, Tennessee. I have made many other pages and sites, but none are as complicated or took as much time as this. I use notepad.exe to write my HTML, so if you're learning yourself you can easily read the indented markup. The periodic table is basically one big HTML table with the borders hidden. It contains no images. You can look at the table with the borders on, if you want to see how it was made. The element descriptions were added later in December 1997 and are from a Franklin Electronic Dictionary.
     I have added an optimized version of the periodic table. It uses Cascading Style Sheets and is compliant with the strict HTML 4 DTD. This was made when no browser fully supported HTML 4. Although a shot in the dark attempt, it should look identical to the periodic table on the main page in the upcoming 5.0 browsers. If your browser does not support HTML 4 and Cascading Style Sheets, the optimized, compliant table may look significantly more distorted or possibly even unviewable in comparison to the browser-compatible table on the main page.
